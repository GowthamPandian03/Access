Sure, here's a simple HTML code to output "Welcome":

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Welcome</title>
</head>
<body>
    <h1>Welcome</h1>
</body>
</html>
```

This code creates a basic HTML document with a heading level 1 (`<h1>`) element displaying the text "Welcome".
